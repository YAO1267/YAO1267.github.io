Interactive Web Marketplace

Developed MokeSell, a user-friendly e-commerce platform for buying and selling new and second-hand items, featuring secure authentication, live chat, and promotional checkout. Designed for trusted, seamless listing and purchasing, with data stored securely in RestDB.

MokeSell

MokeSell is an interactive e-commerce web application designed to connect buyers and sellers in a trusted, user-friendly marketplace for new and second-hand items. The platform focuses on secure transactions, seamless interaction, and an engaging user experience.

Developed a Seller Listing System that allows users to easily upload and manage product listings, including item name, description, price, and image. Each listing displays the seller’s name, promoting transparency and building trust between buyers and sellers.

Designed a Buyer Shopping Experience where users can browse items, add them directly to the cart, or view detailed product pages before making a purchase. Logged-in users can draw up to three vouchers, while visitors are prompted to log in before accessing discount features.

Implemented a Cart and Checkout Module that enables users to apply coupon codes and securely complete payments. Upon successful purchase, the system automatically updates the order status from “Pending” to “Completed,” ensuring clear transaction tracking.

Integrated a Real-Time Chat System using Firebase to support direct two-way communication between buyers and sellers. This feature allows users to negotiate deals, confirm orders, and receive instant updates, enhancing engagement and responsiveness.

Built an Admin Dashboard that visualizes order performance through dynamic bar charts powered by the Chart.js library. This allows administrators to monitor sales trends, completed orders, and platform activity efficiently.

Utilized restdb.io to manage user accounts, authentication, and product information; Firebase Database for real-time messaging; and Lottie animations for smooth transitions, providing an overall modern and interactive interface.
