Healthy Aging Web App

Developed a community-centered health companion platform to empower seniors to age healthily through features such as CC event registration, medication reminders, local service discovery, and location sharing with trusted individuals.

KampungConnect

KampungConnect is a web application designed to help seniors age healthily and navigate daily life with ease by connecting them to a variety of digital features that encourage both online and offline interactions.

Developed an intuitive Medication Management module that allows elderly users to view, add, update, and delete their medication schedules. Each entry includes the medication name, dosage time, and frequency (daily, weekly, etc.), enabling simple and centralized management.

Implemented a Smart Medication Search feature that lets users type or select a medication name from a suggestion list. The system first checks the local database; if data isn’t found, it automatically retrieves details from the openFDA API and stores them locally for future use. If no record exists, a friendly message (“Sorry, this medicine’s information is currently unavailable”) is displayed.

Integrated WhatsApp Reminders to automatically send notifications 30 minutes before each scheduled medication time. Each reminder includes the medication name and dosage time, helping users stay consistent with their prescriptions.

Built a Community Notes system where users can post, update, and delete personal notes or public thoughts. This fosters open communication, feedback sharing, and community engagement among seniors.

Integrated the MeaningCloud Sentiment Analysis API to evaluate the emotional tone of user-submitted notes. Results are visualized as a pie chart on the admin dashboard, showing the proportions of positive, neutral, and negative sentiments. This enables administrators to monitor overall community mood and provide timely support when needed.
